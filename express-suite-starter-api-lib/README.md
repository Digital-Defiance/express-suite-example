# express-suite-starter-api-lib

API business logic library for digital-defiance-express-suite-starter.

## Purpose

Business logic, services, and models for the API layer.

## Structure

```
src/
├── services/       # Business logic services
├── models/         # Data models
├── validators/     # Input validation
└── utils/          # API utilities
```

## Usage

```typescript
import { UserService } from '@express-suite-starter/api-lib';
```
