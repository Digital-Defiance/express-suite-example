# express-suite-example-api-lib

API business logic library for digitaldefiance-express-suite-example.

## Purpose

Business logic, services, and models for the API layer.

## Structure

```
src/
├── services/       # Business logic services
├── models/         # Data models
├── validators/     # Input validation
└── utils/          # API utilities
```

## Usage

```typescript
import { UserService } from '@express-suite-example/api-lib';
```
