## Getting Started

```bash
# Install dependencies
yarn install

# Build all projects
yarn build:dev

# Run development server
yarn serve:dev

# Run tests
yarn test:all
```

## Scripts

- `yarn build` - Production build
- `yarn build:dev` - Development build
- `yarn serve:dev` - Start API server
- `yarn test:all` - Run all tests
- `yarn lint:all` - Lint all projects
